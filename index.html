Project Overview

A web app that turns scattered team knowledge into answers. Ask natural-language questions (e.g., “how do we deploy staging?”) and get citations from source docs, tickets, and past chats. The assistant summarizes meetings and retros, links pipeline failures to related issues, and builds a living knowledge graph across your projects.

AI Q&A Semantic Search Meeting Summaries Knowledge Graph Pipeline Insights RBAC & Audit

Key Features

  • Ask & Answer: Natural-language questions with cited sources.
  • Unified Search: Code, tickets, docs, and chat history in one place.
  • Auto-Summaries: Retros, standups, incident reports and PR threads.
  • Knowledge Graph: Links issues ⇄ commits ⇄ pipelines ⇄ docs.
  • DevOps View: Build failures paired with related work items & owners.
  • Access Control: SSO/OAuth2, roles, and detailed audit logs.
